# 🚀 iide (Intelligent Integrated Development Environment)

[![Vala SDK](https://shields.io)](https://gnome.org)
[![GTK4](https://shields.io)](https://gtk.org)
[![Build System](https://shields.io)](https://mesonbuild.com)
[![License](https://shields.io)](COPYING)

**iide** — это перспективная, ультра-отзывчивая и легковесная нативная среда разработки, написанная на языке **Vala** с использованием передового графического стека **GTK 4**, **Libadwaita** и **Libpanel**. 

Архитектура iide базируется на синергии двух мощных технологий: **Tree-sitter** для мгновенного локального анализа синтаксической структуры и **LSP (Language Server Protocol)** для глубокой асинхронной семантической обработки кода.

## ✨ Ключевые возможности (Features)

### 🌳 Синтаксический анализатор исходного кода (Tree-sitter)
*   **🎨 Синтаксическая подсветка кода:** Сверхбыстрая инкрементальная подсветка синтаксиса на основе честного AST-дерева без использования хрупких регулярных выражений.
*   **📂 Code Folding:** Нативное сворачивание логических блоков кода (классов, функций, циклов).
*   **⌨️ Умные автоотступы (Autoindent):** Вычисление контекстных отступов при переносе строк на основе текущего состояния синтаксического дерева.
*   **🧭 Хлебные крошки структуры кода:** Динамический `BreadcrumbsBar`, отображающий точное положение курсора внутри дерева Tree-sitter-нод (`SourceNodeItem`).

### 🔌 Поддержка LSP
*   **🧩 Параллельная Мульти-LSP Архитектура:** Возможность запуска *нескольких* языковых серверов на один файл параллельно (например, `basedpyright` для анализа типов + `Ruff` для мгновенного линтинга).
*   **💡 Быстрые исправления (Code Actions):** Полноценная поддержка Quick Fixes (`Alt+Enter`). IDE асинхронно опрашивает серверы, объединяет доступные исправления в безрамочное окно и атомарно применяет изменения снизу вверх, защищая координаты строк буфера.
*   **🔍 Инкрементальный Инлайн-Поиск:** Поисковая строка (`FindBar`) встроена в статус-бар холста через `Gtk.Stack` и изолирована в `Gtk.ScrolledWindow`, что полностью исключает баг скачущей ширины `TextView`.
*   **🤖 Автодополнение (Autocomplete):** Нативный сбор контекстных подсказок от LSP-серверов.

### 🎨 Графическая оболочка и Проекты
*   **🔲 Сетка холста (Grid Layout):** Интеграция с `Libpanel` позволяет свободно делить рабочую область на неограниченное количество колонок и строк.
*   **🌗 Нативные темы:** Полная интеграция с Adwaita, поддержка светлого и темного режимов оформления.
*   **📂 Изолированные сессии (.iide):** Состояние вкладок (`workspace.json`), локальные закладки (`bookmarks.json`) и переопределения серверов (`lsp.json`) полностью изолированы в скрытой директории проекта.

## 🛠 Порядок сборки и компиляции

Проект собирается с помощью современной системы сборки **Meson**.

### Системные зависимости
Убедитесь, что в вашей системе установлены `-dev` версии следующих библиотек:
* `valac` (>= 0.56)
* `gtk4`
* `libadwaita-1`
* `gtksourceview-5`
* `libpanel-1` (Libpanel от GNOME Builder)
* `gee-0.8` (Libgee коллекции)
* `json-glib-1.0` (Парсинг и генерация RPC JSON пакетов)

### Сборка проекта
```bash
# Инициализация каталога сборки
meson setup builddir

# Компиляция
meson compile -C builddir

# Запуск среды разработки
./builddir/src/iide
```
