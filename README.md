# iide - ИДЕ

Простая ИДЕ.
Технологичекий стек: vala, GTK-4, tree-sitter

## Features
- Темы: светая/темная
- Синтаксическая подсветка (tree-sitter)
- Folding (tree-sitter)
- Autoindent (tree-sitter)
- Поддержка LSP (python, c, c++, vala)
- Autocomplete (LSP)
